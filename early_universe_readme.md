# Виртуальная Обсерватория Ранней Вселенной

**Цель проекта:**
Создать физически обоснованную, интерактивную симуляцию ранней вселенной, включая этапы **до и после Большого взрыва**, используя открытые данные и модели космологии. Проект ориентирован на **научную визуализацию**, **AI-анализ** и эксперимент с физикой, без физического телескопа.

## Данные

- **Planck CMB**: [https://pla.esac.esa.int](https://pla.esac.esa.int)  
  Данные о космическом микроволновом фоне (температура, флуктуации).  

- **Gaia Catalog**: [https://gea.esac.esa.int/archive/](https://gea.esac.esa.int/archive/)  
  Координаты и свойства миллиардов звёзд.  

- **TLE спутники**: [https://www.celestrak.com/NORAD/elements/](https://www.celestrak.com/NORAD/elements/)  
  Орбиты спутников для динамической визуализации объектов.  

- **IllustrisTNG** (опционально): [https://www.tng-project.org/data/](https://www.tng-project.org/data/)  
  Симуляции эволюции галактик.

## Пошаговый план недели

### День 1: Подготовка среды
- Установить Python, Node.js и библиотеки.  
- Настроить Jupyter/VSCode.  
- Скачивание и первичная обработка Planck CMB и Gaia.

### День 2: Минимальная физическая сетка
- Создание 3D-грид точек → каждая точка = элемент материи.  
- Задание физических параметров: плотность, температура, масса.  
- Визуализация базовой сетки.

### День 3: Инфляция и расширение
- Реализация расширения пространства: scale factor `a(t)` для сетки.  
- Изменение плотности и температуры.  
- Анимация расширения сетки → первые кадры «ранней вселенной».

### День 4: Космический микроволновый фон
- Наложение данных Planck CMB на сетку.  
- Цветовая визуализация флуктуаций температуры.  
- Агенты автоматизируют проверку наложения и генерацию интерактивных карт.

### День 5: Первые атомы и кластеры
- Симуляция рекомбинации (водород, гелий).  
- Формирование первых кластеров → будущие галактики.  
- Вариации физических параметров: плотность, скорость расширения.

### День 6: Гравитационные взаимодействия
- Простая N-body симуляция → формирование кластеров.  
- Визуализация в 3D, интерактивное вращение.

### День 7: Финальный интерактив
- Создание веб-панели: таймлайн, интерактивные объекты, управление параметрами.  
- Сохранение видео/гиф → «путешествие во вселенную до Большого взрыва».

## Промпты для Cursor AI / Antigravity

**Папка:** `prompts/cursor_prompts.md`

#### День 1: Подготовка данных
```
Скачай и очисти данные Planck CMB, Gaia catalog. Преобразуй их в CSV для Jupyter. Сфокусируйся на координатах и физических свойствах (температура, плотность, масса).
```

#### День 2: 3D-сетка ранней вселенной
```
Сгенерируй 3D-грид из 1 миллиона точек с случайным распределением плотности и температуры, сохрани координаты и физические параметры в CSV.
```

#### День 3: Моделирование инфляции
```
Для каждой точки сетки рассчитай движение при экспоненциальном расширении пространства (scale factor a(t)), сохрани координаты на каждом временном шаге.
```

#### День 4: Наложение CMB
```
Примените данные Planck CMB к сетке точек, каждая точка получает значение температуры. Сгенерируй интерактивную визуализацию флуктуаций температуры в 3D.
```

#### День 5: Первые атомы и кластеры
```
Смоделируй рекомбинацию водорода и гелия. Добавь светимость для каждой точки, визуализируй формирование первых скоплений материи.
```

#### День 6: Гравитация и кластеры
```
Смоделируй гравитационное взаимодействие между точками сетки, формирование кластеров и первых мини-галактик. Сохрани кадры для анимации.
```

#### День 7: Финальный интерактив
```
Создай веб-дэшборд с таймлайном ранней вселенной, интерактивной 3D-сеткой и управлением параметрами инфляции, плотности и температуры. Подготовь экспорт видео/гиф.
```

## Расширения и идеи
- VR-погружение → полностью интерактивная вселенная.  
- Альтернативные сценарии: изменяем константы, скорость расширения → смотрим, как меняется структура.  
- AI-классификация необычных паттернов в плотности материи.  
- Подключение спутников TLE → динамика объектов на орбите Земли в реальном времени.

**Примечание:**  
Все расчёты и визуализации опираются на открытые данные и физические модели. Проект можно адаптировать под обучение, демонстрацию или развлечение, но при этом он максимально физически обоснован.

Создан стартовый Jupyter Notebook с первой 3D-сеткой ранней вселенной и подключением данных CMB. Он включает:
- генерацию случайной физической сетки;
- подключение температуры CMB для каждой точки;
- визуализацию с Plotly;
- опциональную визуализацию через PyVista;
- сохранение данных для дальнейшей работы и интеграции в проект.
- Следующим шагом агенты могут заполнить реальными данными Planck и расширить сетку до миллиона точек для полноценной симуляции.
