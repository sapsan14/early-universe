## Реализованные модули

### 1. Решатель Фридмана (`friedmann.py`)

**Что это:** Решает первое уравнение Фридмана — центральное уравнение космологии, описывающее расширение Вселенной.

**Основное уравнение:**

$$ H^2(a) = H_0^2 \left[ \frac{\Omega_r}{a^4} + \frac{\Omega_m}{a^3} + \frac{\Omega_k}{a^2} + \Omega_\Lambda \right] $$

Каждый член представляет компоненту энергетического бюджета Вселенной:
- $\Omega_r / a^4$ — **излучение** (фотоны + нейтрино): разбавляется быстрее всех, т.к. помимо уменьшения числовой плотности ещё и красное смещение удлиняет длину волны
- $\Omega_m / a^3$ — **материя** (барионы + тёмная материя): разбавляется при расширении
- $\Omega_k / a^2$ — **кривизна**: геометрический вклад (для плоской Вселенной $\Omega_k = 0$)
- $\Omega_\Lambda$ — **тёмная энергия**: плотность не меняется при расширении (космологическая постоянная)

**Реализованные функции:**

| Функция | Описание | Формула |
|---------|----------|---------|
| `E_squared(a)` | Безразмерная функция Хаббла $E^2 = H^2/H_0^2$ | см. выше |
| `hubble(a)` | Параметр Хаббла H(a) [км/с/Мпк] | $H = H_0 \sqrt{E^2(a)}$ |
| `hubble_from_z(z)` | H(z) через красное смещение | $a = 1/(1+z)$ |
| `deceleration_parameter(a)` | Параметр замедления q(a) | $q = (\Omega_r/a^4 + \Omega_m/(2a^3))/E^2 - \Omega_\Lambda/E^2$ |
| `cosmic_time(a)` | Космическое время t(a) [Гыр] | $t = \int_0^a \frac{da'}{a' H(a')}$ |
| `conformal_time(a)` | Конформное время η(a) [Мпк] | $\eta = \int_0^a \frac{c \, da'}{a'^2 H(a')}$ |
| `comoving_distance(z)` | Сопутствующее расстояние χ(z) [Мпк] | $\chi = c \int_0^z \frac{dz'}{H(z')}$ |
| `luminosity_distance(z)` | Расстояние светимости [Мпк] | $d_L = (1+z) \chi$ |
| `angular_diameter_distance(z)` | Угловое расстояние [Мпк] | $d_A = \chi / (1+z)$ |
| `age_of_universe()` | Возраст Вселенной [Гыр] | $t_0 = t(a=1)$ |
| `matter_radiation_equality()` | Эпоха равенства материя-излучение | $a_{eq} = \Omega_r / \Omega_m$ |
| `evolve_scale_factor()` | Эволюция a(t) через интегрирование ODE | $da/dt = aH(a)$ |
| `solve_background()` | Полное решение фоновой космологии | Все величины из конфига |

**Ключевые термины:**

- **Масштабный фактор a(t)**: безразмерная мера размера Вселенной. $a = 1$ сегодня, $a < 1$ в прошлом. Все расстояния масштабируются как $a$.
- **Параметр Хаббла H(a)**: скорость расширения. $H_0 = 67.36$ км/с/Мпк — текущее значение ("постоянная Хаббла").
- **Красное смещение z**: наблюдаемая величина, связанная с a: $1 + z = 1/a$. Фотон, испущенный при $z = 1100$, растянут в 1101 раз.
- **Конформное время η**: время, в котором световой конус — прямая линия на пространственно-временной диаграмме. Определяет каузальную структуру.
- **Сопутствующее расстояние χ**: расстояние с поправкой на расширение. "Сколько далеко объект *сейчас*."
- **Расстояние светимости d_L**: расстояние по яркости (закон обратных квадратов). $d_L = (1+z)^2 d_A$ (соотношение Этерингтона).

**Валидация (14 тестов):**
- $H(a=1) = H_0$ ✓
- Возраст Вселенной $t_0 = 13.80$ Гыр (Planck: 13.797 ± 0.023) ✓
- $\chi(z=1100) \approx 14.2$ Гпк ✓
- $d_L / d_A = (1+z)^2$ (соотношение Этерингтона) ✓
- $z_{eq} \approx 3450$ (Planck: 3387) ✓
- Вселенная ускоряется сегодня ($q_0 < 0$), замедлялась в прошлом ($q > 0$) ✓

### 2. Рекомбинация (`recombination.py`)

**Что это:** Вычисляет долю свободных электронов $x_e(z)$ — ключевую величину, определяющую, когда Вселенная стала прозрачной для фотонов.

**Физика процесса:**

При $z > 1500$ ($T > 4000$ К) вся Вселенная — горячая плазма: протоны + электроны + фотоны в тепловом равновесии. При охлаждении электроны рекомбинируют с протонами:

$$p + e^- \leftrightarrow H + \gamma$$

Когда $x_e$ падает достаточно низко, фотоны перестают рассеиваться и начинают свободно распространяться — это **поверхность последнего рассеяния** ($z \approx 1100$, $T \approx 3000$ K). Эти фотоны мы наблюдаем как CMB.

**Реализованные модели:**

#### Уравнение Саха (равновесное приближение)

$$ \frac{x_e^2}{1 - x_e} = \frac{1}{n_b} \left(\frac{m_e k_B T}{2\pi\hbar^2}\right)^{3/2} e^{-E_I / k_B T} $$

- Точно при $T \gg 3000$ K (полная ионизация) и $T \ll 3000$ K (полная нейтральность)
- **Ломается** во время перехода: предсказывает слишком быструю рекомбинацию
- Причина: не учитывает, что расширение Вселенной выводит систему из равновесия

**Техническая деталь:** использована численно устойчивая форма квадратной формулы $x_e = 2S / (S + \sqrt{S^2 + 4S})$ для избежания catastrophic cancellation при больших S (высокие температуры).

#### Уравнение Пиблса (неравновесная ODE)

$$ \frac{dx_e}{dt} = C_r \left[\beta(T)(1 - x_e) - \alpha^{(2)}(T) n_b x_e^2 \right] $$

- $\alpha^{(2)}(T)$ — коэффициент рекомбинации Case-B (только на возбуждённые уровни)
- $\beta(T)$ — скорость фотоионизации (из детального баланса)
- $C_r$ — фактор коррекции Пиблса, учитывающий:
  - **Двухфотонный распад** 2s-состояния (скорость $\Lambda_{2s} = 8.227$ с$^{-1}$)
  - **Космологический уход** Лайман-альфа фотонов из резонансной линии (расширение красносмещает фотоны)

**Почему Case-B:** прямая рекомбинация на основной уровень ($n = 1$) излучает фотон с энергией $\geq 13.6$ эВ, который немедленно ионизирует соседний атом. Поэтому учитываются только рекомбинации на $n \geq 2$.

**Дополнительно реализовано:**

| Функция | Описание |
|---------|----------|
| `optical_depth(z)` | Оптическая глубина $\tau(z) = \int n_e \sigma_T c \, dt$ |
| `visibility_function(z)` | Функция видимости $g(z) = \|d\tau/dz\| e^{-\tau}$ — вероятность последнего рассеяния |

**Ключевые термины:**

- **Ионизационная доля x_e**: $x_e = n_e / n_H$. $x_e = 1$ — полностью ионизован, $x_e \ll 1$ — нейтральный водород.
- **Case-B рекомбинация**: рекомбинация, исключающая прямые переходы на основной уровень (их фотоны реабсорбируются).
- **Оптическая глубина τ**: мера непрозрачности. $\tau \gg 1$ — фотоны не могут пройти (tight coupling). $\tau \ll 1$ — свободное распространение.
- **Функция видимости g(z)**: узкий пик при $z \sim 1100$ — "когда именно" образовался CMB.
- **Freeze-out**: рекомбинация "замерзает", не доходя до $x_e = 0$ (остаточная ионизация $x_e \sim 10^{-4}$), потому что расширение уменьшает n_b быстрее, чем электроны успевают рекомбинировать.

**Валидация (5 тестов):**
- $x_e \approx 1$ при $T = 50\,000$ K ✓
- $x_e \approx 0$ при $T = 1\,000$ K ✓
- $x_e$ начинается вблизи 1 при $z = 1800$ ✓
- $x_e < 0.1$ при $z \approx 800$ ✓
- Остаточная ионизация $x_e > 0$ при $z = 200$ (freeze-out) ✓

### 3. Иерархия Больцмана (`boltzmann.py`)

**Что это:** Система связанных ODE, описывающая эволюцию возмущений плотности всех компонент Вселенной (фотоны, барионы, тёмная материя, нейтрино) в фурье-пространстве.

**Физическая картина:**

Вселенная не идеально однородна — в ней есть малые флуктуации плотности (1 часть из 100 000). Эти возмущения эволюционируют:
- **Гравитация** стягивает материю в сгустки
- **Давление фотонов** противодействует сжатию
- **Расширение Вселенной** разбавляет всё

Баланс этих сил создаёт **акустические осцилляции** — звуковые волны в первичной плазме. Их следы видны как пики в спектре CMB.

**Мультипольное разложение фотонов:**

Распределение фотонов раскладывается по мультиполям $\Theta_l(k, \tau)$:
- $l = 0$: **монополь** — флуктуация температуры / сверхплотность
- $l = 1$: **диполь** — скорость (допплеровский сдвиг)
- $l = 2$: **квадруполь** — анизотропное напряжение (порождает поляризацию)
- $l \geq 3$: высшие мультиполи (свободное распространение)

**Реализованные уравнения:**

| Компонента | Уравнения | Физика |
|-----------|-----------|--------|
| Фотоны $\Theta_l$ | Иерархия мультиполей + томсоновское рассеяние | Давление + связь с барионами |
| Поляризация $\Theta_P$ | Иерархия поляризации | Квадруполь → поляризация |
| CDM $(\delta_c, v_c)$ | Континуальность + Эйлер (без давления) | Только гравитация |
| Барионы $(\delta_b, v_b)$ | Континуальность + Эйлер + томсоновское трение | Давление + связь с фотонами |
| Нейтрино $N_l$ | Как фотоны, но без рассеяния | Свободное распространение с $z \sim 10^{10}$ |
| Метрика $(\Phi, \Psi)$ | Уравнение Пуассона | Гравитационные потенциалы |

**Ключевые термины:**

- **Tight coupling**: при $z \gg 1100$ фотоны и барионы ведут себя как единая жидкость (скорость рассеяния $\gg$ скорость расширения).
- **Free streaming**: после декаплинга фотоны распространяются свободно; высшие мультиполи $l \gg 1$ начинают заполняться.
- **Конформный ньютоновский калибр**: выбор координат, где метрика имеет вид $ds^2 = a^2[-(1+2\Psi)d\tau^2 + (1-2\Phi)dx^2]$. $\Phi$ и $\Psi$ — потенциалы Бардина.
- **Адиабатические начальные условия**: все компоненты начинают с коррелированными возмущениями, заданными одним скалярным полем (инфлатоном).

### 4. Теория возмущений (`perturbations.py`)

**Что это:** Вычисление наблюдаемых спектров — передаточная функция T(k), спектр мощности материи P(k), и угловой спектр CMB $C_l$.

**Передаточная функция T(k):**

Показывает, как каждая фурье-мода возмущения изменилась от начальных условий до сегодняшнего дня. Кодирует всю пост-инфляционную эволюцию: переход материя-излучение, BAO, Силк-демпфинг.

Реализованы две аппроксимации:

| Формула | Точность | Физика |
|---------|----------|--------|
| **BBKS** (Bardeen-Bond-Kaiser-Szalay 1986) | ~15% | Только CDM, без BAO |
| **Eisenstein-Hu 1998** | ~5% | CDM + барионы, BAO, Силк-демпфинг |

**Спектр мощности материи P(k):**

$$ P(k) = P_{prim}(k) \cdot T^2(k) \cdot k $$

где $P_{prim}(k) = A_s (k/k_*)^{n_s - 1}$ — примордиальный спектр от инфляции.

P(k) — статистическое описание "бугристости" Вселенной: сколько структуры на каждом масштабе.

**Реализованные функции:**

| Функция | Описание |
|---------|----------|
| `transfer_function_eh98(k)` | Передаточная функция Эйзенштейна-Ху |
| `transfer_function_bbks(k)` | Передаточная функция BBKS |
| `primordial_power_spectrum(k)` | Примордиальный спектр $P_s(k) = A_s(k/k_*)^{n_s-1}$ |
| `matter_power_spectrum(k)` | Полный спектр $P(k) = P_{prim} T^2 k$ |
| `growth_factor(z)` | Фактор роста D(z), нормализованный к D(0) = 1 |
| `sigma_R_squared(R)` | Дисперсия $\sigma^2(R) = \int P(k) W^2(kR) k^2 dk$ |
| `angular_power_spectrum_sw(l)` | Упрощённый $C_l$ (приближение Сакса-Вольфа) |
| `compute_matter_pk(k)` | P(k) из конфига одной строкой |

**Ключевые термины:**

- **Передаточная функция T(k)**: "фильтр", который Вселенная применяет к начальным возмущениям. $T(k \to 0) = 1$ (большие масштабы не затронуты), $T(k \gg k_{eq})$ падает (малые масштабы подавлены).
- **BAO (барионные акустические осцилляции)**: осцилляции в T(k) от звуковых волн в ранней плазме. "Стандартная линейка" космологии (~150 Мпк).
- **Силк-демпфинг**: затухание мелкомасштабных возмущений из-за диффузии фотонов (фотоны "размывают" возмущения при свободном распространении).
- **σ₈**: среднеквадратичные флуктуации плотности в сферах $R = 8$ Мпк/h. Мера "кластеризованности" Вселенной. Planck: $\sigma_8 = 0.811$.
- **Фактор роста D(z)**: как быстро растут возмущения. $D \propto a$ в эпоху материи, замедляется при $\Lambda$-доминировании.

**Валидация (12 тестов):**
- $T(k \to 0) \approx 1$ ✓
- $T(k)$ убывает при больших k ✓
- $P_{prim}$ = const при $n_s = 1$ (масштабная инвариантность) ✓
- $P(k) > 0$ везде ✓
- Пик P(k) при $k \sim 0.01$–$0.02$ h/Мпк ✓
- $D(z=0) = 1$ ✓

### 5. Сферические гармоники (`spherical_harmonics.py`)

**Что это:** Обёртка вокруг `healpy` для работы с CMB-картами в формате HEALPix.

**Физика:**

Температура CMB на небе $T(\theta, \phi)$ раскладывается по сферическим гармоникам — аналогу ряда Фурье на сфере:

$$ T(\theta, \phi) = \sum_{l=0}^{\infty} \sum_{m=-l}^{l} a_{lm} Y_{lm}(\theta, \phi) $$

Коэффициенты $a_{lm}$ содержат всю информацию. Их дисперсия — угловой спектр мощности:

$$ C_l = \langle |a_{lm}|^2 \rangle $$

**HEALPix** (Hierarchical Equal Area isoLatitude Pixelization) — стандартная схема пикселизации сферы, используемая Planck, WMAP и всеми CMB-экспериментами.

**Реализованные функции:**

| Функция | Описание |
|---------|----------|
| `map_to_alm(map)` | Разложение карты → коэффициенты $a_{lm}$ |
| `alm_to_map(alm, nside)` | Синтез карты из $a_{lm}$ |
| `map_to_cl(map)` | Оценка $C_l$ из карты |
| `generate_cmb_map(cl, nside)` | Генерация случайной реализации CMB из теоретического $C_l$ |
| `generate_alm_from_cl(cl)` | Генерация $a_{lm}$ из $C_l$ без карты |
| `nside_to_npix(nside)` | Количество пикселей: $N_{pix} = 12 \cdot N_{side}^2$ |
| `get_pixel_area(nside)` | Площадь одного пикселя [стерадиан] |
| `angular_distance(θ₁, φ₁, θ₂, φ₂)` | Угловое расстояние на сфере |

**Ключевые термины:**

- **Мультиполь l**: угловой масштаб на небе. $l \sim 180°/\theta$. $l = 1$ — диполь (движение наблюдателя), $l \sim 200$ — первый акустический пик, $l > 2000$ — мелкие масштабы.
- **$a_{lm}$**: комплексные коэффициенты разложения. Для гауссова поля (стандартная модель) $a_{lm} \sim \mathcal{N}(0, C_l)$.
- **$C_l$**: угловой спектр мощности. Главная наблюдаемая CMB. Содержит информацию обо всех космологических параметрах.
- **nside**: разрешение HEALPix. $N_{side} = 64$ (быстрые тесты, ~49k пикселей), $N_{side} = 2048$ (разрешение Planck, ~50M пикселей).

### 6. Модели инфляции (`inflation.py`)

**Что это:** Вычисление предсказаний различных моделей космологической инфляции.

**Физика:**

Инфляция — период экспоненциального расширения в первые $\sim 10^{-32}$ секунды. Решает три проблемы Большого Взрыва:
- **Проблема горизонта**: почему CMB одинаков во всех направлениях (области, которые "не должны" были общаться, были связаны до инфляции)
- **Проблема плоскостности**: почему $\Omega_k \approx 0$ (инфляция "разглаживает" кривизну)
- **Проблема монополей**: почему мы не видим магнитных монополей (они были разбавлены)

Квантовые флуктуации инфлатонного поля растягиваются до космических масштабов и становятся зёрнами всех структур.

**Slow-roll параметры:**

$$ \epsilon = \frac{M_{Pl}^2}{2}\left(\frac{V'}{V}\right)^2 \ll 1, \quad \eta = M_{Pl}^2 \frac{V''}{V} \ll 1 $$

Определяют наблюдаемые:
- Скалярный спектральный индекс: $n_s = 1 - 6\epsilon + 2\eta$
- Отношение тензор/скаляр: $r = 16\epsilon$

**Реализованные модели:**

| Модель | Потенциал | $n_s$ (N=55) | $r$ (N=55) | Статус |
|--------|-----------|-------------|-----------|--------|
| **Starobinsky R²** | $(3/4)M_{Pl}^2 m^2[1-e^{-\sqrt{2/3}\phi/M_{Pl}}]^2$ | 0.964 | 0.004 | Согласуется с Planck ✓ |
| **Квадратичный** | $m^2\phi^2/2$ | 0.964 | 0.145 | r выше верхнего предела ✗ |
| **Натуральная** | $\Lambda^4[1+\cos(\phi/f)]$ | зависит от f | зависит от f | Допустимо при f >> M_Pl |
| **Степенной** | $V_0(\phi/M_{Pl})^p$ | зависит от p | зависит от p | p=1 допустимо |

**Ключевые термины:**

- **Инфлатон**: скалярное поле $\phi$, движущая сила инфляции. Медленно скатывается по потенциалу $V(\phi)$.
- **Slow-roll**: приближение, при котором кинетическая энергия поля мала по сравнению с потенциальной ($\epsilon \ll 1$, $|\eta| \ll 1$).
- **N e-folds**: количество раз, которое Вселенная удвоилась в размере. Требуется $N \gtrsim 50$–60 для решения проблем горизонта и плоскостности.
- **Тензорные моды (r)**: гравитационные волны от инфляции. Обнаружение $r > 0$ через B-mode поляризацию CMB — "дымящийся пистолет" инфляции. Planck+BICEP: $r < 0.036$.
- **Модель Старобинского R²**: возникает из $f(R) = R + R^2/(6M^2)$ гравитации. Лучшее согласие с данными Planck.

**Валидация (9 тестов):**
- $n_s$ Старобинского совпадает с Planck ($0.955 < n_s < 0.975$) ✓
- $r$ Старобинского мало ($< 0.01$) ✓
- Квадратичная модель предсказывает больший $r$, чем Старобинского ✓
- Больший $p$ → больший $r$ ✓

## Известные ограничения

1. **Иерархия Больцмана** реализована как набор RHS-функций, но полный интегратор (line-of-sight для C_l) — задача Фазы 1b или валидации через CLASS.
2. **Передаточная функция** — аналитическая аппроксимация EH98 (~5% точность). Для < 0.1% нужна валидация против CLASS (задача `class-validation`).
3. **Угловой спектр C_l** — только Сакс-Вольф приближение (большие масштабы). Полный C_l с акустическими пиками требует полной интеграции Больцмана.
4. **Рекомбинация** — Forward Euler интегратор. Для production-качества рекомендуется RK4 или adaptive solver.
5. **Все модули CPU-only** — JIT-компиляция на GPU включается автоматически при наличии `jaxlib` с CUDA.
